  INSTALL LARAVEL

composer global require "laravel/installer"



  CREATE A NEW PROJECT

laravel new <ProjectName>

cd <ProjectName>
php artisan app:name <ProjectName>
php artisan key:generate


  CREATE LOCAL DOMAIN FOR YOUR APP

Create a VirtualHost in Apache's httpd.conf

Create an alias for that VirtualHost in Window's hosts file in Windows\System32\drivers\etc



  CONFIGURE DATABASE (MYSQL)

All the settings are in /.env


    CREATE A DATABASE

mysql -uroot -p
create database hireme;
use hireme;
show tables;



  CREATE MODEL, CONTROLLER AND MIGRATION

php artisan make:model <NameSingular> -mc


    CREATE THEM MANUALLY

php artisan help make:model
php artisan make:controller <NamePlural>controller
php artisan make:model <NameSingular>
php artisan make:migrate create_<NamePlural>_table --create=<TableName>


    CREATE A AUTO-FILLED CONTROLLER (BOILERPLATE)

php artisan make:controller <NamePlural>Controller -r



  ROUTES

GET     /route              index()
GET     /route/create       create()
POST    /route              store(Request $request)
GET     /route/{id}         show($id)
GET     /route/{id}/edit    edit($id)
PATCH   /route/{id}         update(Request $request, $id)
DELETE  /route/{id}         destroy($id)



  PROECTION AGAINST MASS ASSIGNMENT IN POST REQUESTS

Create a Model that inherits from Eloquent/Model assigning $guarded

The rest of the Models in the application must inherits from this new Model



  MIGRATION

    MIGRATE

php artisan migrate


    RE-MIGRATE (RESET AND RE-RUN)

php artisan migrate:refresh


    ROLLBACK MIGRATIONS

php artisan migrate:reset


    REFRESH COMPOSER DATABASE

composer dump-autoload
